About the Bot

Joel MD is a multipurpose WhatsApp bot designed for automation, media tools, entertainment, and group management. Built with ❤️ by joeljamestech.

Compatible with WhatsApp Multi-Device Beta.

Features

AI-powered replies, translation, moderation tools, media downloads, anime info, meme generation, sticker maker, voice effects, text-to-speech, PDF tools, group controls, NSFW toggle, image manipulation, and more.

Popular Commands

/menu

Displays full command list.

/sticker

Converts image/video to sticker.

/play [songName]

Downloads YouTube audio.

/anime [title]

Shows anime info via Jikan API.

/meme

Sends a random meme.

/lyrics [song]

Fetches lyrics of the song.

/vcf [text]

Converts contacts into a downloadable PDF.

/say [type]

Converts text to funny voice (e.g., robot, deep, chipmunk).

/ai [prompt]

Chat with AI powered by GPT-4.

/kick @user

Removes a member from the group (admin only).

/nsfw on/off

Toggles NSFW features (admin only).
